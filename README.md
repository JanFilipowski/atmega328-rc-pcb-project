# ATmega328 RC PCB Project

A custom KiCad hardware design for an ATmega328-based RC electronics board.

## Project Overview

This repository contains the complete KiCad design files for a PCB built around the ATmega328 microcontroller.
It includes schematic, board layout, project settings, and backup snapshots.

## Repository Contents

- `ATMEGA328ip.pcb.kicad_sch` - Main schematic file.
- `ATMEGA328ip.pcb.kicad_pcb` - PCB layout.
- `ATMEGA328ip.pcb.kicad_pro` - KiCad project definition.
- `ATMEGA328ip.pcb.kicad_prl` - Local project settings.
- `ATMEGA328ip.pcb-backups/` - Automatic backup copies.

## Requirements

- KiCad 6.x or newer (recommended: KiCad 7/8).

## How To Open The Project

1. Start KiCad.
2. Open `ATMEGA328ip.pcb.kicad_pro`.
3. Use **Schematic Editor** to inspect the circuit.
4. Use **PCB Editor** to inspect traces, footprints, and board outline.
